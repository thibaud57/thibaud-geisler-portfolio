# Thibaud Geisler Portfolio

Plateforme personnelle servant de vitrine professionnelle et de hub central
(services, projets, démos externes) en IA, développement full-stack et formation.

## Stack

Next.js 16 · React 19 · TypeScript 6 · Tailwind 4 · shadcn/ui · PostgreSQL 18 · Prisma 7 · Dokploy

## Getting Started

```bash
pnpm install
pnpm dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).
